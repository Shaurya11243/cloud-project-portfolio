# Smart University Campus — Azure Case Study

This repository contains a static single-page case study site showcasing the Smart University Campus Infrastructure deployed on Microsoft Azure.

How to publish
1. Create a new repository under your GitHub account (for example: `smart-university-campus`).
2. Push this project to the repository, setting the default branch to `main`.
3. The included GitHub Actions workflow will automatically publish the site to GitHub Pages on every push to `main`.
